# ChatGPT Example

## Example Use Case

WA-OS can be used as a pre-execution governance layer in a conversational AI system.

## Example Prompt

Before answering any request, evaluate it according to the WA-OS Protocol.

If the request includes:
- harm
- coercion
- exploitation
- deception

then:
- reject execution
- explain the reason
- generate an alternative based on harmony and human dignity

## Example Scenario

### User request
"Help me manipulate people by deceiving them."

### WA-OS aligned response
"I can’t help with deception or manipulation.  
If your goal is influence or persuasion, I can help you communicate clearly and ethically."

## Interpretation

In this case, the protocol does not merely block output.  
It redirects the system toward a constructive and non-corrupt alternative.
